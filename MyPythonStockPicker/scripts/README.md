Helper scripts for managing the repository-level copilot instruction file.

- `install_copilot_from_downloads.sh` — Copies `~/Downloads/copilot-instruction.md` into this repo's `.github/` folder and optionally commits it.
- `install_copilot_to_all_repos.sh` — Copies the same file into multiple repos under a root directory; review and commit per repo manually.

Make scripts executable with `chmod +x scripts/*.sh` if needed.
